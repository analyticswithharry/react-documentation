# React Beginner Lab

A simple React project for learning by running and changing code.

## What is inside

- `src/pages/` — lesson pages
- `src/lessons/basics/` — JSX, props, state, forms
- `src/lessons/hooks/` — `useEffect`, `useRef`, `useContext`, `useReducer`
- `src/lessons/projects/` — mini project examples
- `src/context/` — shared context example
- `src/hooks/` — custom hook example

## How to run

1. Open a terminal in `react-beginner-lab`
2. Install packages with `npm install`
3. Start the app with `npm run dev`
4. Open the local URL shown in the terminal

## Best way to learn

1. Open one file from `src/lessons/`
2. Change one value or one function
3. Save the file
4. Check the browser
5. Repeat until the concept feels easy

## Real examples to study

- `src/lessons/real/ProductFilterDemo.jsx` — search + filter like a shop or dashboard
- `src/lessons/real/FetchUsersDemo.jsx` — real API fetch with loading and error states
- `src/lessons/real/CartReducerDemo.jsx` — cart logic with `useReducer`

## How to learn with code

Try these edits one by one:

1. Change the product names in `ProductFilterDemo.jsx`
2. Add a new category option
3. Change the API URL or limit in `FetchUsersDemo.jsx`
4. Add a quantity button in `CartReducerDemo.jsx`
5. Rebuild the same feature without copying
